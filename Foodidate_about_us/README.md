# preprocessor PUG Sass


##On the shell
```
    git clone [githubrepo](#https://github.com/yung0512/Foodidate/tree/main/Foodidate_about_us.git)

    npm i     
    //install dependencies (pug,sass)

    npm run compile:sass
    //to compile sass file

    npm install pug-cli --save-dev
    //install 
    npm run compile:pug
    //compile .pug to .html

    npm install -g live-server
    //start the html file automatically(install globally)

    live-server
   //start the about us page


```
